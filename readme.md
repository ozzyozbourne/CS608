## Running the Program

### Requirements:

- Java (version 17 or higher)
- Any IDE or command line tool with Java support

### Steps to Run:

1. Clone the repository:

   ```bash
   git clone https://github.com/your-repository-url
   ```

2. CD into CS608/java/src

   ```bash
   cd CS608/java/src
   ```

3. Compile the Java program:

   ```bash
   javac <FILENAME>.java
   ```

4. Run the program:

   ```bash
   java <FILENAME>
   ```
